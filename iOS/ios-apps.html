Secret Apps

iOS, just like any other operating system, has lots of stuff you don't usually see. In the past, Apple has revealed occasional hidden apps, and I've made sure to get my hands on some.

How to find hidden apps

The official Shortcuts app is one of the most powerful tools you'll find in a normal iOS system. It allows you to create functions and scripts that automate your process after it starts. However, the Shortcuts app has also seemed to expose certain hidden apps in the past, and even in some newer iOS versions. The way it works, is you can use the "Open App" script, which will ask you to choose an app to open. In the list of apps, some hidden ones will be shown.

The Feedback app was possible to access before it was patched somewhere around iOS 18. Since it is no longer possible to access, I've created an iCloud link that you can use to access this hidden app. Feedback (iOS hidden app). I'd be careful with this one as it is seems to be one of the more developer-oriented apps.

The Code Scanner app is another somewhat-hidden app. You can access it through the control center widget or add a shortcut.

The Field Test app is, once again, only somewhat hidden. You can either use a shortcut or dial *3001#12345#*

If you know of any other hidden apps, make sure to let me know in Pull requests.

Ghost Apps

These are generally associated with Apple's built-in wallpaper, customization, or system assets. These are "Color," "Emoji," "Kaleidoscope," "Pride," and "Unity." There could be more, or maybe they're just unique to my device. If you have an idea, tell me in Pull requests. These are the only ones that appear in the Shortcuts menu, as other "ghost apps" exist and appear in Screen Time settings, such as "Screenshots," "User Authentication," or "ScreenTimeUnlock."
